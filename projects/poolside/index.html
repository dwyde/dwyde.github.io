Poolside

2011-05-20

Poolside was my thesis project at New College of Florida, accompanied by a paper.

It was an online prototype to allow others to review and reproduce data analyses. Poolside supported both Python and Ruby back in 2011, along with visualizations, though it was otherwise less polished than the IPython/Jupyter notebook.

The code is on GitHub.
